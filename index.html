Sistema de Agendamento

Sistema de cadastro de docentes, discentes e técnicos-administrativo para agendar consultas com os psicólogos da UEMA. Sistema desenvolvido com PHP, Laravel, HTML, CSS, JavaScript, consumindo API, servidor de email, PostgreSQL e Docker.

Participei do desenvolvimento, teste, treinamento e manutenção. O código é privado da instituição.

SEDS 2.0

Sistema interno de coleta, análise, tratamento, avaliação e integração de grande quantidade de dados para as remessas das carteiras estudantis dos alunos da UEMA. Desenvolvido com PHP, Laravel, HTML, CSS e PostgreSQL consumindo 2 bancos (o interno e o de pesquisa).

Participei de todas as etapas, desde a coleta de requisitos até a manutenção do sistema. O código é privado da instituição.
